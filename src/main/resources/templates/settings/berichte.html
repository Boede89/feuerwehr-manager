Einsatzberichte — Divera-Anbindung

Status-Bezeichnungen werden unter Admin → Schnittstellen → DIVERA 24/7 → Status-ID gepflegt. Nur Personal mit den ausgewählten Rückmeldungen wird später automatisch in den Bericht übernommen.

Mehrfachauswahl — nur diese Rückmeldungen werden für die automatische Personal-Übernahme genutzt.

Noch keine Status-IDs hinterlegt — zuerst unter Admin → Schnittstellen → DIVERA → Status-ID anlegen.

Einstellungen für diesen Berichtstyp folgen in einer späteren Version.
