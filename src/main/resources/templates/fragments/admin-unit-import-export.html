Datenbank-Backups können nur vom Superadmin erstellt und wiederhergestellt werden.

Datenbank sichern (Export)

Erstellt ein vollständiges SQL-Backup der gesamten Datenbank (alle Einheiten, Benutzer, Personal, Atemschutz, Fahrzeuge, Einstellungen usw.). Das Schema selbst wird über Flyway verwaltet — gesichert werden nur die Daten.

Backup herunterladen
Datenbank wiederherstellen (Import)

Nur Backups aus dem Feuerwehr-Manager. Max. Dateigröße: 128 MB

Alle vorhandenen Daten werden gelöscht und durch den Inhalt der Backup-Datei ersetzt. Dieser Vorgang kann nicht rückgängig gemacht werden. Vorher ein aktuelles Backup erstellen.
